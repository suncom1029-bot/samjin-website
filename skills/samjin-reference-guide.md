# SAMJINLND 홈페이지 개발 Skill

## 개요
이 Skill은 SAMJINLND 홈페이지 제작 프로젝트에서 일관된 방향과 기준을 유지하기 위해 만들어졌습니다.

**기반 문서**: `skills/reference.md` (종합 레퍼런스)

---

## 🎯 이 Skill을 사용하는 시기

이 Skill은 SAMJINLND 홈페이지 관련 다음 작업을 할 때 참조하세요:

### 1️⃣ 콘텐츠 개발
- 페이지 문구 작성
- 섹션 구성 기획
- 이미지 선택 및 방향성 결정

### 2️⃣ 디자인 결정
- 색상 팔레트 선택
- 레이아웃 구성
- 브랜드 톤 유지

### 3️⃣ 기술 구현
- 페이지 구조 설계
- 컴포넌트 분류
- 메타데이터 설정

### 4️⃣ 검수 및 QA
- 콘텐츠 정확성 확인
- 브랜드 일관성 점검
- 정보 완성도 검증

## 📋 참조 포인트

### 1. 기본 회사 정보
```
회사명: ㈜삼진엘앤디 / SAMJIN L&D
설립: 1987년 1월 6일 (38년 역사)
본사: 경기도 화성시 동탄기흥로 64-17
규모: 800명 임직원, 963억원 매출(2025)
웹: www.samjin.co.kr
```

### 2. 핵심 메시지 (강제)
- **메인**: "세상을 바꾸는 기술, 삼진엘앤디"
- **서브1**: "AI 데이터센터 핵심 부품 기업"
- **서브2**: "38년 정밀 금형 기술의 정수"
- **서브3**: "글로벌 네트워크, 로컬 전문성"

### 3. 3대 설립이념 (반드시 포함)
1. 고객에게 최고의 만족을 드릴 수 있는 품질
2. 공존공영의 기본 정신과 신뢰
3. 창조와 도전으로 신나는 일터

### 4. 사업 분야 (우선순위)
1. **2차전지 부품** ⭐⭐⭐ (최우선 강조 - AI 시대)
   - 가스켓, ESS, UPS 배터리 부품
   - 삼성SDI 협력, 북미 시장 공략
   
2. **디스플레이 부품** ⭐⭐ (기존 사업)
   - 도광판, 확산판, Mold Frame
   
3. **OA 기기** ⭐ (보조 사업)
   - 복합기, Finisher

4. **금형·자동화** ⭐ (기술 기반)
   - Hot Runner, 자동화 시스템

### 5. 5대 핵심 기술 (필수)
1. **금형 기술**: Hot Runner 특허 5건
2. **광학 기술**: Roll Stamp, 패터닝 기술
3. **배터리 부품**: IATF16949, 현대/삼성 협력
4. **AI 비전 기술**: Deep Learning 기반 검사 시스템
5. **글로벌 제조**: 한국, 멕시코, 베트남 거점

### 6. 글로벌 거점
| 지역 | 법인명 | 역할 | 인원 |
|------|-------|------|------|
| 한국 | SAMJIN LND | 본사 & 개발 | 130명 |
| 멕시코 | COMEX PLATECH | 제조 & 판매 | 80명 |
| 베트남 | SAMJIN LND VINA | 배터리부품 제조 | 590명 |
| 미국 | ELEMEK, INC. | 영업 거점 | 소수 |

### 7. 최신 전략 (2026년)
- **AI 데이터센터 시장 진입**
  - ESS·UPS 배터리 부품 확대
  - 북미 시장 본격 공략
  - 2026년 레코드 매출 목표
  
- **제조 자동화 (AX) 전환**
  - ERP 기반 통합 관리
  - 공정 자동화 고도화
  
- **신규 사업**
  - 수소 저장탱크 (2026.6 인증 목표)

### 8. 주요 수상 (신뢰도)
- 삼성전자 상생협력 금상
- World Class 300 선정
- LED 차세대 조명전시회 대상
- 뿌리기술 명가 대통령상
- 품질경쟁력 우수기업

### 9. 문화 & 인사
- 직급: 프로(단일 호칭), 능력 기반
- 복리후생: 기숙사, 스포츠센터, 경조금
